# Use last-import-wins for Word edit conflicts

The demo allows multiple active **Word編集セッション** and concurrent Web edits for the same **Webドキュメント**, resolving conflicts by applying the last successful **終了取り込み** without an overwrite confirmation. This intentionally avoids merge and conflict-confirmation logic so the demo can focus on OneDrive-mediated Word editing, accepting that earlier Web or Word edits may be overwritten.
